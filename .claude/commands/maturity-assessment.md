# /maturity-assessment

Run an Intelligence Maturity Model assessment for a system.

## Instructions

When this command is invoked:

1. Ask the user to describe their current AI system
2. Assess against the 5 IMM levels from `maturity/model.md`
3. Identify which level they are at and why
4. List what is needed to reach the next level
5. Recommend which AIEL phases they should prioritise

Output a structured assessment report.
