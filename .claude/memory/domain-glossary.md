# AIEL Domain Glossary

**AIEL** — Aether Intelligence Engineering Lifecycle. The methodology framework.

**Intelligence System** — Any system that perceives, reasons, remembers, and acts. Distinguished from software by persistent memory and autonomous decision-making.

**Cognitive Architecture** — The structural design of how a system reasons, remembers, and communicates. Includes memory types, reasoning patterns, and agent topology.

**Agent Contract** — The formal specification of an agent: its capabilities, authority boundaries, tool access, communication protocols, and confidence thresholds.

**Confidence Gate** — The rule that any agent decision with confidence < 0.8 requires human-in-the-loop review before enforcement.

**Memory Engineering** — The discipline of designing what an intelligence system remembers, how long it remembers it, how memories are reinforced, and how they decay.

**Knowledge Engineering** — The discipline of designing what an intelligence system knows, how knowledge is ingested, validated, and kept current.

**Evaluation Engineering** — The discipline of measuring intelligence quality across dimensions: correctness, hallucination rate, memory recall, decision quality, safety, cost, and latency.

**Governance Engineering** — The discipline of ensuring human oversight, audit trails, policy enforcement, and compliance across all agent decisions.

**Learning Engineering** — The discipline of ensuring the intelligence system improves over time through feedback ingestion and memory reinforcement.

**Evolution Engineering** — The discipline of planning how an intelligence system grows new capabilities without breaking existing behaviour.

**Intelligence Maturity Model (IMM)** — Five-level model from Prompt Experiments (Level 1) to Distributed Intelligence Ecosystems (Level 5).

**EEIK Bootstrap** — Enterprise Engineering Intelligence Kit. The governance layer applied to every Aether repository: CLAUDE.md, memory files, agents, hooks, commands.

**Persistent Intelligence** — An intelligence system that retains memory, identity, and learned behaviour across interactions and restarts. Contrasted with stateless AI.
