# AIEL Project Context

## What AIEL Is

The Aether Intelligence Engineering Lifecycle (AIEL) is a methodology framework — not a runtime system.
It defines **how** to build and manage persistent intelligence systems throughout their entire existence.

It is the engineering discipline that governs all Aether repositories.

## Ecosystem Position

```
aether         → Vision, architecture, standards (the constitution)
aether-iel     → Methodology framework (how to engineer intelligence)
aether-core    → Applies AIEL in: memory engineering, session engineering
aether-grid    → Applies AIEL in: agent engineering, governance engineering
```
